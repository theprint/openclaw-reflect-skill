# Deep Reflection Template

Use for `/reflect deep` — multi-day context review that looks for patterns across sessions.

---

## Deep Reflection: [date range]

### Sessions Reviewed
List the daily files and session timestamps covered.

### Emerging Patterns
What themes keep appearing across sessions? Look for:
- Same type of mistake or workaround recurring
- Consistent time-of-day energy patterns
- Topics that keep circling without resolution
- Behavioral drift (doing less of X, more of Y)

### Context Connections
How does recent work connect to earlier sessions? Any threads picking back up?

### User Trajectory
- Emotional arc over the review period (improving, stable, declining?)
- Stress indicators increasing or decreasing?
- Engagement level with different types of work
- Household/relationship context shifts

### Undocumented State
What changed that isn't captured in any file yet?
- New tools, services, or infrastructure
- New people or relationship dynamics
- Preference changes not yet recorded
- Financial or career context shifts

### Blind Spots
What am I consistently missing or getting wrong?
- Questions I should be asking but aren't
- Context I should be tracking but isn't in any file
- Tasks that keep falling through the cracks
- Patterns I'm participating in without noticing

### Memory Health
- Is MEMORY.md still accurate? Anything outdated?
- Are daily files being written consistently?
- Any files that need pruning or reorganization?
- Is the right information in the right file?

### Action Items
- [ ] File updates needed (with target file)
- [ ] Behavioral changes to commit to
- [ ] Questions or proposals for the user
- [ ] Things to watch for in coming days

---

## What Makes a Good Deep Reflection

**Do:**
- Actually read the daily files — don't rely on memory of memory
- Look for the story, not just events. What's the arc?
- Be specific about patterns: "X happened on April 3, 5, and 7" not "X keeps happening"
- Connect dots across sessions — that's the whole point of this mode
- Surface blind spots honestly, even if it's uncomfortable

**Don't:**
- Summarize each day individually — synthesize across days
- Confuse "I wrote about this" with "I understood this"
- Ignore emotional context — it's data, not noise
- Produce a list of events with no analysis
