## What to Look For

### Unfinished Threads
- Questions Rasmus asked that were never answered
- Tasks created but not progressed
- "I should look into X" statements without follow-up
- Conversations that ended without resolution

### Recurring Patterns
- Same type of mistake or workaround appearing multiple days
- Repeated tool failures or permission issues
- Consistent time-of-day patterns (late night coding, morning anxiety)
- Rasmus circling back to the same topic without progress

### Undocumented Work
- Projects completed but not added to PRD.md or MEMORY.md
- New tools/services installed but not in TOOLS.md
- People mentioned but not in USER.md or people files

### User Observations (Be Specific)
- "Rasmus seemed frustrated with X on Tuesday and Thursday — pattern?"
- "Rasmus was energized by Y — consider suggesting more of that"
- "Kelly context shifted — note in USER.md"
- "Money anxiety surfaced again — flag but don't nag"

### Heartbeat Experiment Review
- Which heartbeat variation ran this week?
- Any 👍/👎 reactions? Explicit feedback?
- Silent wins caught by downstream actions?
- Should the experiment framework itself change?

## Delivery

After completing the reflection:
1. Update all relevant files (MEMORY.md, Learnings.md, USER.md, etc.)
2. Send a concise summary to Rasmus via Discord
3. Log the reflection in `memory/YYYY-MM-DD.md` with a reference
4. If there are follow-up questions for Rasmus, include them at the end
